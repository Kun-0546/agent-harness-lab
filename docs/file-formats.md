## program

PM 给 coding agent 的实验指令,一个实验一份。位置:`experiments/<编号-名字>/program.md`。

结构:

```
# 实验 <编号-名字> · program

## 假设
<这次实验想验证什么>

## 声明
- 环境:<被测环境是什么 + 取的初始状态;无环境写"无">
- 对话模式:<模拟 / 回放 / 固定>
- 状态:<累积 / 重置>
- 评分:<评分器类型(规则脚本 / LLM Judge / 组合)+ 打分粒度>
- 运行模式:<人评 / 自迭代>

## 留/丢规则
<一个改动满足什么算"留下",否则回滚>

## 喊人规则
<coding agent 跑到什么情况停下、交回 PM>
```

- **假设、声明** —— 每个实验都填。
- **留/丢规则、喊人规则** —— 只在运行模式 = 自迭代时填,人评模式留空;出厂带预置。
- **状态** —— 累积:上一轮进化后的状态 = 下一轮评测的初始状态。重置:每轮回到初始。

例子待补:晚点脱敏构造一个,不用真实实验。
